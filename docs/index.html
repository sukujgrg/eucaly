Projection Workflow

  1. Select an item from Library, Playlist, Audio, Web, or Windows.
  2. Preview it without changing what is currently projected.
  3. Load or switch Preview into Current when ready.
  4. Project Current to the selected display.

Use File > New Lyrics or Command-N to create a new lyrics file. Use File > Quick Open or Command-Shift-O to open the command palette and search the library without leaving the keyboard.

What eucaly Supports

Lyrics

Create lyrics from File > New Lyrics or Command-N, then write plain text with a title, section headings, and verse lines.

Media

PDFs, images, videos, webpages, and live app-window capture for flexible presentation sources.

Layers

Independent slide visibility, background visuals, background audio, and timer or clock overlays.

Background Audio

Import audio or video files into the library, choose a playable item from Audio, then control play, stop, loop, seek, and volume separately from slides.

Search

Use the command palette to quickly search the library, open webpages, create lyrics, and jump to supported files without browsing folders manually.

Writing Lyrics

Save lyrics as a plain text file. Section headings can be written with or without brackets, and blank lines separate blocks.

  • Supported headings are Intro, Verse, Chorus, Pre-Chorus, Post-Chorus, Bridge, Instrumental, Vamp, Coda, Ending, Outro, and Tag.
  • Common aliases are Vers and Strophe for Verse; Refrain and Hook for Chorus; and Pre or Prechorus for Pre-Chorus.
  • Headings like [Chorus], [ Chorus ], and Chorus: are accepted.
  • Hook is treated as a chorus heading.
  • The first standalone line is treated as the song title. If there is no title line, the file name is used.
  • Direct CCLI/SongSelect lyric exports are supported, including common trailing copyright, CCLI song number, and license metadata.
  • The Format button can normalize common headings after you paste or type lyrics.

Getting Help

  • Use GitHub Issues for bugs, crashes, and feature requests.
  • Include your macOS version, eucaly version, and the file type involved.
  • For projection issues, mention whether a second display or projector was connected.
  • For lyrics formatting issues, include a small sample of the text file.
